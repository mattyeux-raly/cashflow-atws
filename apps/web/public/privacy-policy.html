4. Durées de conservation

  • Données de compte : durée du contrat + 1 an
  • Transactions comptables : 3 ans glissants (configurable par le cabinet)
  • Journaux d'audit : 5 ans (obligation légale)
  • Consentements : durée du contrat + 5 ans (preuve)
  • Analytics : 13 mois maximum

6. Vos droits

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d'accès (Art. 15) : obtenir une copie de vos données via le Centre de données personnelles
  • Droit de rectification (Art. 16) : modifier vos informations via les Paramètres
  • Droit à l'effacement (Art. 17) : demander la suppression de votre compte et données
  • Droit à la portabilité (Art. 20) : exporter vos données en JSON/CSV
  • Droit d'opposition (Art. 21) : désactiver la synchronisation, supprimer vos données
  • Droit de retirer votre consentement à tout moment

Pour exercer vos droits : dpo@cashflow-app.fr ou via le Centre de données personnelles de l'application.

7. Sécurité

  • Chiffrement TLS 1.3 pour les données en transit
  • Chiffrement AES-256 pour les tokens Pennylane au repos
  • Isolation des données multi-tenant via Row Level Security
  • Authentification sécurisée via Supabase Auth
  • Journalisation de toutes les actions sensibles

8. Cookies

L'application utilise uniquement des cookies techniques strictement nécessaires à son fonctionnement (session d'authentification). Aucun cookie publicitaire ou de tracking n'est déposé sans votre consentement explicite.
